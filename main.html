Why am I learning software development?

Well, I decided that I want to become a full-stack Data Scientist. The field of data science for business is quite new and a lot of people still try to define it but through my experience I came up with my own full-stack Data Scientist definition in my head. Let me try and define it shortly:

A data scientist is someone with business knowledge who can collect, clean, and organise big data with the end goal of mining them and provide business value from the data.
A machine learning engineer (MLE) is someone with the technical knowledge required to develop a software product/app which uses machine learning algorithms insight. Often, MLEs will work with DSs who are responsible to provide the problem definitions the MLE tries to solve.

Having the skills to both define the problems we can or cannot solve with data and being able to develop an end-to-end software product/app which utilises machine learning algorithms to solve the problems I define will allow me to develop data solutions that provide solutions at scale for one or multiple organisations.
